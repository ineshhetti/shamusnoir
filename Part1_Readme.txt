Key Features of the Code

    Tempo: Set to 120 BPM (Allegro).
    Dynamics:
        ff (fortissimo) for loud hits.
        p (piano) for softer hits.
        pp (pianissimo) for very soft notes.
    Accents: Adjusted velocity levels to reflect the > articulation.
    Crescendo: Gradual increase in velocity over the third measure.
    Rhythm: Sixteenth-note patterns across all measures.

How to Use

    Save the script as a .py file.
    Run it on your machine after installing midiutil (pip install midiutil).
    Open the resulting snare_drum_allegro.mid file in any MIDI player or music software to hear the snare drum pattern.
